Introduction:

An electronic voting system is a web-based application for casting voting online. It aims to make the voting process easier, faster, and more accessible. The project aims to develop a simple electronic voting system for colleges and schools. The system will ensure accuracy, security, and transparency of the voting process. The system will provide an easy and convenient way for students to cast their votes.

About :

This web-base application is bulit for the purpose of creating a good enviroment for Voting . In this application there are two Roles i.e. Voter and Candidate where Candidate login in the page to Register them for election where Voter login just to vote to their Favourite Candidate and more importantly you can see live results.

This application is bulit by Ajay Thapa and Kaushal Bhuju. We use Html and Css to design the interface of the website and JavaScript for Form validation , Php as our Backend Language and Mysql for connecting to Database in this website. We have used Visual Studio Code (Vs code). We have used various sources and help of our teachers for creating this website.
